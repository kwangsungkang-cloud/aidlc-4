# 테이블오더 서비스 — 컴포넌트 정의

---

## 1. 백엔드 컴포넌트 (api-server / Spring Boot)

### 1.1 AuthComponent
- **목적**: 인증 및 권한 관리
- **책임**:
  - 테이블 태블릿 자동 로그인 처리
  - 매장 관리자 JWT 인증
  - 슈퍼 관리자 JWT 인증
  - 토큰 발급/검증/갱신
  - 로그인 시도 제한 (brute-force 방지)
- **관련 요구사항**: FR-01, FR-06, FR-10

### 1.2 StoreComponent
- **목적**: 매장 정보 관리
- **책임**:
  - 매장 CRUD
  - 매장별 설정 관리
- **관련 요구사항**: FR-06, FR-10

### 1.3 TableComponent
- **목적**: 테이블 및 세션 관리
- **책임**:
  - 테이블 등록/설정
  - 테이블 세션 생성/종료
  - 이용 완료 처리 (주문 이력 이동, 리셋)
- **관련 요구사항**: FR-01, FR-08

### 1.4 MenuComponent
- **목적**: 메뉴 및 카테고리 관리
- **책임**:
  - 메뉴 CRUD
  - 카테고리 관리
  - 메뉴 노출 순서 관리
  - 메뉴 이미지 S3 업로드
- **관련 요구사항**: FR-02, FR-09

### 1.5 OrderComponent
- **목적**: 주문 처리 및 관리
- **책임**:
  - 주문 생성
  - 주문 상태 변경 (대기중→준비중→완료)
  - 주문 삭제
  - 주문 내역 조회 (현재 세션/과거 이력)
  - SSE 실시간 주문 이벤트 발행
- **관련 요구사항**: FR-04, FR-05, FR-07, FR-08

### 1.6 AdminComponent
- **목적**: 관리자 계정 관리
- **책임**:
  - 매장 관리자 계정 CRUD
  - 슈퍼 관리자 계정 관리
  - 매장별 관리자 할당
  - 활동 이력 기록 및 조회
- **관련 요구사항**: FR-10, FR-11

### 1.7 SSEComponent
- **목적**: 실시간 이벤트 스트리밍
- **책임**:
  - SSE 연결 관리 (매장별)
  - 주문 이벤트 브로드캐스트
  - 연결 상태 모니터링
- **관련 요구사항**: FR-07

### 1.8 FileStorageComponent
- **목적**: 파일 업로드/관리
- **책임**:
  - S3 이미지 업로드
  - 이미지 URL 생성
- **관련 요구사항**: FR-09

## 2. 고객용 프론트엔드 컴포넌트 (customer-web / React)

### 2.1 AuthModule
- 자동 로그인 처리, 로컬 저장소 관리, 세션 상태 관리

### 2.2 WelcomeModule
- 초기 접속 안내 화면, 환영 메시지, 사용법 안내 (FR-01-1)

### 2.3 MenuModule
- 카테고리별 메뉴 조회, 메뉴 카드 표시, 카테고리 네비게이션

### 2.4 CartModule
- 장바구니 관리, 수량 조절, 총 금액 계산, 로컬 저장

### 2.5 OrderModule
- 주문 생성, 주문 확인, 주문 내역 조회

### 2.6 SplitBillModule
- 분할 계산 (메뉴별/인원별), 클라이언트 측 계산

---

## 3. 관리자용 프론트엔드 컴포넌트 (admin-web / React)

### 3.1 AuthModule
- 관리자 로그인, JWT 토큰 관리, 세션 유지

### 3.2 DashboardModule
- 테이블별 그리드 레이아웃, 실시간 주문 모니터링 (SSE), 신규 주문 강조

### 3.3 OrderManagementModule
- 주문 상태 변경, 주문 삭제, 주문 상세 보기

### 3.4 TableManagementModule
- 테이블 초기 설정, 이용 완료 처리, 과거 주문 내역 조회

### 3.5 MenuManagementModule
- 메뉴 CRUD, 이미지 업로드, 노출 순서 관리

### 3.6 AdminManagementModule (슈퍼 관리자 전용)
- 매장 관리자 계정 CRUD, 활동 이력 조회
